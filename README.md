# Phonebook-WebApp

## Summary
An app that displays a phonebook table—sorted by name—and allows for adding new contacts via a form.

## Usage Instructions
1. Change into the app directory: `cd phonebook-app`
2. Run `npm install`
3. Run `npm start`
4. Visit http://localhost:3000/ in your browser
